# Simple-SoC-Design-using-Vivado-IP-Integrator
Designed and implemented a basic System-on-Chip (SoC) using Xilinx Vivado IP Integrator. The project integrates a MicroBlaze soft processor with AXI GPIO using AXI SmartConnect, along with local memory, clock generation, processor reset logic, and debugging support. The design was validated successfully and prepared for FPGA implementation.

#Features
MicroBlaze Soft Processor
AXI SmartConnect
AXI GPIO Interface
Local BRAM Memory
Clock Wizard
Processor System Reset
Block Design Validation
Ready for Bitstream Generation

# Tools Used
Xilinx Vivado
IP Integrator
AXI Protocol
FPGA Design Flow
